# Airbnb-Clone

## Overview
This project is a simplified Airbnb-style accommodation listing web application built as part of the ProDev Frontend Engineering Program.
The goal is to demonstrate mastery of modern frontend development tools, scalable design patterns, and real-world application architecture.

The app allows users to:

- Browse listings
- View property details
- Search/filter locations
- Log in/sign up
- Create/manage listings

## Tech Stack
Framework & Languages
- Next.js 14 (App Router)
- TypeScript
- React
- Tailwind CSS

Key Concepts Practiced
- Routing & dynamic routes
- API integration
- State management
- Component architecture
- Responsive UI with TailwindCSS
- Deployment workflow
- System design principles
- Reusable UI components
- Fetching server & client data
